Our Products

Intelligent Document Processing for KFintech

AssemblyTech developed a robust OCR-based solution for KFintech to automate data extraction from structured and semi-structured banking forms. The system intelligently parses critical information such as customer name, PAN card details, and account numbers, storing them securely in a relational database.

Key Technologies: OCR (Tesseract, EasyOCR), FastAPI, PostgreSQL, PDF parsing, Celery, Docker

YouTube Copyright Automation for ZeeTV

We engineered a proprietary automation tool for ZeeTV's digital content team to monitor and detect copyright-infringing uploads across YouTube. The tool identifies unauthorized content, initiates takedown procedures, and ensures ZeeTV's channels are monetization-compliant.

Impact: Enabled ZeeTV to restore ad monetization status and safeguard intellectual property.
Key Technologies: YouTube Data API, Selenium, Node.js automation, Google Cloud Functions

GenAI PDF Intelligence Platform for MLX (pdfocr.ai)

For MLX (pdfocr.ai), we built a Generative AI-powered PDF Intelligence System that parses large volumes of unstructured PDFs, extracts key metadata and content, stores embeddings in a Vector Database, and performs semantic search. Users can ask natural language questions and receive contextually accurate responses generated by LLMs.

Use Case: Legal document analysis, financial report summarization, and compliance tracking.
Key Technologies: LangChain, OpenAI LLM, FAISS/ChromaDB, pdfplumber, FastAPI, PostgreSQL
